# Politica - Privacidade, Retencao e LGPD

## Objetivo
Orientar o tratamento de dados pessoais e sensiveis do ambiente escolar.

## Regras Sugeridas
- coletar apenas dados necessarios ao caso de uso
- acesso a dados pessoais deve respeitar papel e tenant
- consultas sensiveis devem ser auditaveis
- exportacao de dados deve ser controlada
- retencao e descarte devem seguir politica institucional e legal
- mascaramento deve ser considerado para telas e logs quando apropriado

## Testes Recomendados
- ator autorizado acessa dado sensivel
- ator nao autorizado recebe negacao
- log nao expone dado pessoal desnecessario
- exportacao inclui trilha de auditoria
