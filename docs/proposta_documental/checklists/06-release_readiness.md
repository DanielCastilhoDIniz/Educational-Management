# Checklist - Release Readiness

- [ ] Suite automatizada esta verde
- [ ] Mudancas de schema foram validadas
- [ ] Alertas e logs relevantes foram revisados
- [ ] Erros novos possuem mapeamento operacional conhecido
- [ ] Documentacao do caso de uso foi atualizada
- [ ] Politicas afetadas foram revisadas
- [ ] Riscos e rollback foram discutidos
- [ ] Time de suporte sabe o que mudou
