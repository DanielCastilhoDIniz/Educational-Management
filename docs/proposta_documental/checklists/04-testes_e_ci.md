# Checklist - Testes e CI

- [ ] Testes de dominio cobrem invariantes alteradas
- [ ] Testes de Application cobrem fluxo e traducao de erros
- [ ] Testes de infraestrutura cobrem persistencia relevante
- [ ] Suite completa passa localmente
- [ ] CI reproduz o ambiente de testes necessario
- [ ] Dependencias de teste estao no requirements correto
- [ ] Cobertura minima continua atendida
- [ ] Nao ha teste fragil dependente de mensagem textual de banco sem justificativa
